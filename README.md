## Overview
**Statics** is a feature-rich Valorant companion app for Android, offering a seamless experience to track, manage and share your Valorant stats, store and loadouts conveniently on mobile. With over 50,000 downloads, Statics stands out as a reliable, all-in-one tool for Valorant enthusiasts — built with a modern Material You design and translated into 20+ languages.

### Features of Statics

#### Account & Experience
| Feature | Description |
|---|---|
| Material You Design | Built on Material 3 Expressive, with a per-account accent colour automatically themed from your equipped player card. |
| Riot Sign-On | Secure official Riot login with silent session re-auth so you stay signed in. |
| Multi-Account Support | Manage and switch between multiple Valorant accounts effortlessly. |
| Multi-Language Support | Translated into 20+ languages for accessibility. |
| Dark & Light Mode | Choose between dark and light themes for optimal viewing. |
| Offline Mode | Browse your cached stats, loadouts and store even without a connection. |
| Customisable Start | Pick which screen the app opens on, plus a right-hand mode for one-handed use. |
| Wear OS Companion | A companion watch app with a rank tile and at-a-glance stats. |

#### Live Match & Party
| Feature | Description |
|---|---|
| Live Game Tracking | See agent select and in-game lobbies live — teammates' and enemies' ranks, stats and loadouts. |
| Party Tracking | View your party members in real time, with built-in party chat and pings. |
| Pre-Game Intel | Inspect every player in agent select before the match begins. |

#### Stats & Match Analysis
| Feature | Description |
|---|---|
| Comprehensive Stat Tracking | Monitor your performance across all game modes. |
| Match History | Access up to 100 past matches, stored locally for quick retrieval. |
| Visual Match Analysis | Detailed scoreboard, economy and round-by-round breakdowns. |
| Interactive Minimap | Round-by-round replay of player positions, spike plants and kills on the map. |
| Match Insights | Per-map kill/death heatmaps and your most-played recent teammates. |
| MMR & Rank History | Track rank progression with win/loss history and your all-time peak rank. |
| Weapon Rankings | Personal and global weapon stats and rankings. |
| Career Profiles | View your own and other players' public career profiles. |
| Playtime Stats | See how much time you've put into the game. |

#### Store & Cosmetics
| Feature | Description |
|---|---|
| Full Store | Daily shop, featured bundles, Night Market and accessories — all in one place. |
| Wishlist & Alerts | Wishlist skins and get notified when they appear in your shop. |
| Store & Accessory History | Look back over your past daily shops and accessory rotations. |
| Battlepass | Track your battlepass tiers and progress. |
| Loadout & Skin Presets | Save full loadouts (weapons, player card, title, sprays), randomise them, and share showcases. |
| Cosmetics Browser | Browse skins, agents, sprays, cards, titles and gun buddies with owned/locked indicators. |

#### Social
| Feature | Description |
|---|---|
| Statics Social | A community hub with a global feed, chat/forum and Looking-for-Group (LFG). |
| User Profiles | Personal and public profiles to share with others. |
| Friends List | Riot friends via XMPP, with PC vs Console differentiation, filtering and whisper chat. |
| Global Chat System | Interact with other users worldwide. |

#### Tools
| Feature | Description |
|---|---|
| Lineups Library | Browse utility and ability lineups with video clips. |
| Game Database | Reference data for all agents, maps and weapons. |

#### Home Screen Widgets & Shortcuts
| Feature | Description |
|---|---|
| Store Widgets | Daily Shop, Bundle, Night Market and Accessory widgets. |
| Friends Widget | See online friends at a glance, with a PC/Console toggle and friend-online alerts. |
| Rank & Win-Rate Widgets | Keep your current rank and recent win-rate on your home screen. |
| Last Match Widget | Your most recent match's result and performance. |
| Quick Settings Tile & Shortcuts | A Quick Settings tile to check your shop, plus dynamic launcher shortcuts. |

### Description
Statics provides a user-friendly interface for accessing Valorant statistics directly from your Android device. Key features include seamless Riot Sign-On integration, comprehensive stat and match tracking, a full in-app store with purchasing, loadout management, a social hub, home-screen widgets and a Wear OS companion. Designed with the aesthetic of Valorant in mind, Statics offers both functionality and style for avid players.
